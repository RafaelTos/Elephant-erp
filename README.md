## 📋 Sobre o Projeto
O **Elephant ERP** é um sistema de planejamento de recursos empresariais minimalista desenvolvido em PHP nativo e banco de dados MySQL. O projeto foi estruturado com foco em boas práticas de segurança, mantendo credenciais de conexão e chaves de ambiente isoladas e protegidas.

### 🚀 Funcionalidades Principais
- [x] Conexão segura e permanente com banco de dados utilizando PDO.
- [x] Arquitetura modular preparada para expansão de painéis (dashboards).
- [x] Proteção nativa de credenciais e arquivos sensíveis utilizando controle de versões do Git.

---

## 🛠️ Tecnologias Utilizadas
- **Linguagem Principal:** PHP (Back-End)
- **Banco de Dados:** MySQL / MariaDB
- **Ambiente Local:** XAMPP (Apache)
- **Versionamento:** Git & GitHub
